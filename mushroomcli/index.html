📖 Installation Guide

1. Download Meteor Client from meteorclient.com
2. Place MushroomCLI.jar in your .minecraft/mods/ folder
3. Launch Minecraft with Fabric + Meteor installed
4. Press .mushroom or find in Meteor GUI

⚙️ Features

▪️ AutoMushroom - Automatically collect/grow mushrooms
▪️ MushroomESP - Highlight mushroom blocks through walls
▪️ SporeTrail - Leave particle trail behind you
▪️ MushroomAura - Auto-attack entities near mushrooms
▪️ More features coming soon...
